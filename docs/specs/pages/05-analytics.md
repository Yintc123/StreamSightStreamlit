# 頁面規格:資料分析

- 頁面編號:5
- 對應模組:模組 4 資料分析
- 存取權限:已登入使用者

## 目的

對歷史資料做統計、聚合與趨勢分析,並可下載 Excel。

## 版面

以 `st.tabs` 分為三個分頁:統計 / 趨勢 / 匯出。

## UI 版面規劃

寬版,`st.tabs(["統計", "趨勢", "匯出"])`;三分頁共用頂部篩選列(時間範圍 + 分類)。

```
時間範圍[起]-[迄]   分類[▾]                             ← 共用篩選 st.columns

── 統計 ────────────────────────────────────────────────
 ┌────────┬────────┬────────┬────────┐
 │ 總計    │ 平均    │ 最大    │ 最小    │                ← st.columns(4)+st.metric
 └────────┴────────┴────────┴────────┘
 分類聚合表(groupby)  ← st.dataframe

── 趨勢 ────────────────────────────────────────────────
 粒度: (時)(日)(週)   ← st.radio/segmented
 時間序列折線圖 + 分類疊圖   ← st.line_chart / Plotly

── 匯出 ────────────────────────────────────────────────
 目前篩選:{摘要}
 [ ⬇ 下載 Excel (.xlsx) ]   [ ⬇ 下載 CSV ]   ← st.download_button
```

| 分頁 | 主要元件 | 備註 |
|---|---|---|
| 篩選(共用) | `st.date_input` + `st.selectbox` | 三分頁一致;建議 `st.cache_data` |
| 統計 | `st.columns(4)` + `st.metric` + `st.dataframe` | 聚合用 pandas `agg`/`groupby` |
| 趨勢 | `st.radio`/`segmented_control` 選粒度 + `st.line_chart` | 分類疊圖比較 |
| 匯出 | `st.download_button` | 內容與畫面篩選一致;無資料時 `disabled` |
| 空狀態 | `st.info` | 範圍內無資料時停用匯出 |

## 功能細節

### 統計
- 篩選:時間範圍(`st.date_input`)、分類。
- 統計量:總計、平均、最大、最小(pandas `agg`)。
- 分類聚合:依分類 `groupby` 呈現彙總表。

### 趨勢
- 時間序列折線圖(可選彙總粒度:時 / 日 / 週)。
- 分類疊圖比較。

### 匯出
- 依目前篩選條件,以 API 取回的資料在前端產生 Excel(`openpyxl` + `BytesIO`),或呼叫後端匯出端點。
- `st.download_button` 下載 `.xlsx`。
- 可選:同時提供 CSV。

## 資料

- 讀取:透過後端**分析 API**(依時間範圍與分類篩選)取得統計 / 聚合結果;前端不直接連 DB。
- 唯讀頁面。

## 狀態與錯誤處理

- 範圍內無資料 → 空狀態提示,停用匯出。
- 大範圍查詢 → 建議快取(`st.cache_data`)。

## 依賴 / 備註

- 匯出內容需與畫面篩選一致。
